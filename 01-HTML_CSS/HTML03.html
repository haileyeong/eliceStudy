움직이는 웹사이트 제작

Transform

  • rotate

    - rotate(angle) : 입력한 각도만큼 회전하며, 음수도 입력 가능

  • scale

    - scale(x, y) : 숫자는 비율을 의미하며 width를 x배, height를 y배 만큼 확대

  • skew

    - skew(x, y) : x축, y축을 기준으로 입력한 각도만큼 비튼다

  • translate

    - translate(x, y) : 선택한 오브젝트의 좌표 변경

  • prefix 접두사

    - Transform은 CSS의 최신 버전에서만 실행 가능한 키워드이지만,

    prefix를 추가하면 하위 버전의 브라우저에서도 실행 가능

Transition

  • property

    - property : 효과를 적용하고자 하는 css 속성

  • duration

    - duration : 효과가 나타나는데 걸리는 시간

  • timing-function

    - timing-function : 효과의 속도 (linear는 '일정하게'라는 의미)

  • delay

    - dealy : 특정 조건 하에서 효과 발동 (1s는 '1초 후'라는 의미)

  • hover

    - 가상 선택자:hover

    css에서 미리 만들어 놓은 클래스 -> "마우스를 올렸을 때"라는 조건.
