Cascading Style Sheet

  • 정보(HTML)와 디자인(CSS)의 분리
  • 문서의 레이아웃과 스타일 정의
  • HTML로 작성된 정보를 꾸며주는 언어

CSS 구성 요소

  • 선택자
    • 디자인을 적용할 HTML 영역
  • 속성
    • 어떤 디자인을 적용할지 정의
  • 속성값
    • 어떤 역할을 수행할지 구체적으로 명령. 세미콜론(;) 필수 입력.

CSS 연동 방법 세 가지

  1. Inline Style Sheet
    2. 태그 안에 직접 원하는 스타일을 적용
  2. Internal Style Sheet
    3. style 태그 안에 넣기
  3. External Style Sheet

    4. link 태그로 불러오기

    html, css 각각의 문서 안에서 따로 관리하여 상대적으로 가독성이 높고 유지보수가 쉬움

선택자(Selector)

- HTML의 어떤 요소에 CSS를 적용할 것인가.

  • Type - 타입
  • Class - 클래스 (태그 별명) . 온점
  • ID - 아이디 (태그 이름) # 샵

부모 자식 관계

- 특정 태그는 반드시 포함하는 관계를 가진다. = 부모 자식 관계

- 원하는 지역에만 css 속성을 적용하기 위해 부모를 구체적으로 표기

캐스케이딩

CSS의 우선순위를 결정하는 세 가지 요소

  1. 순서
    2. - 나중에 적용한 속성값의 우선순위가 높음
  2. 디테일
    3. - 더 구체저기으로 작성된 선택자의 우선순위가 높음
  3. 선택자
    4. - style > id > class > type 순으로 우선순위가 높음

CSS 주요 속성

width, height
  • width 속성

    선택한 요소의 넓이를 설정

    고정값(%), 가변값(px)

  • height 속성

    선택한 요소의 높이를 설정

  • font

    font-family

    - 브라우저마다 지원하는 폰트가 다름
    - 입력한 순서대로 우선순위 적용.
    - sans-serif는 마지막에 작성하는 디폴트 값.

    font-weight

    100~900 사이의 숫자를 입력할 수도 있음.

  • border

    border-style

    실선: solid
    점선: dotted

    주석과 같이 한 줄에 이어 쓸 수도 있음.
    이때, 쉼표는 작서하지 않고 띄어쓰기만 함.

  • background

    background-repeat

    x축으로 반복: repeat-x
    y축으로 반복: repeat-y
    반복하지 않음: no-repeat

    background-position

    공간 안에서 이미지의 좌푤르 변경할 때.
    top, bottom, center, left, right 등.
